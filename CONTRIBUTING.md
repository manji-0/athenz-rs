# Contributing

Thanks for helping improve `athenz-provider-tenant`.

## Development setup

- Rust toolchain (stable)
- Optional: `rustfmt`, `clippy`

## Build and test

```sh
cargo build
cargo test
```

## Formatting and linting

```sh
cargo fmt
cargo clippy
```

## Documentation updates

- Keep `README.md` short and link to `docs/` for details.
- Each new doc should include at least one minimal example.

## Example test invocation

```sh
cargo test policy::tests
```
